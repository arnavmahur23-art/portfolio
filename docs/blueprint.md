# **App Name**: Arnav.IO

## Core Features:

- Animated Hero Section: Display a captivating hero section with animated typography and an interactive cursor effect.
- About Me Section: Showcase a split-layout 'About Me' section with smooth motion-on-scroll animations.
- Interactive Project Cards: Design interactive project cards that expand on hover and support video previews of work.
- Skills and Tools Display: Create a dynamic display of skills and tools with animated icons in a timeline format.
- Contact/Support CTA: Implement a minimal, glowing call-to-action button for contact and support.
- Parallax Background: Layer backgrounds to create parallax depth effect

## Style Guidelines:

- Primary color: Soft white (#F0F4F8) for a clean, modern aesthetic.
- Background color: Dark cinematic gray (#222222) to provide a premium, high-tech feel and excellent contrast for other elements; very desaturated; dark scheme
- Accent color: Neon blue (#64B5F6) for highlights and interactive elements, drawing inspiration from advanced UI designs.
- Font: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body. Note: currently only Google Fonts are supported.
- Use minimalist icons that are animated on scroll, for a futuristic feel.
- Employ a split layout design for the 'About Me' section.
- Integrate glassmorphism panels to sections in order to make them appear modern and standout from the dark background.
- Implement cursor-follow interactive effects with a soft glowing orb, enhancing the visual experience. Physics-based animations, subtle and delayed motion will create an added sense of depth.
